本機 Agent 設定

            

            

                
「本機」模式會把測試交給本機的 local_agent.py 執行，會 自動開啟有頭 Chromium，你可以在本機視覺化觀察整個測試過程。

                
    
                    
  1. 點下方「下載 Agent」取得 local_agent.py。
    2. 
                    
  2. 一次性安裝依賴（在本機終端機）：
    
                        pip install playwright requests
    
                        playwright install chromium
                    
    3. 
                    
  3. 啟動 agent（保持常駐）：
                        
    
                            
                            
                        
    
                    
    4. 
                    
  4. 切換頂部「環境：本機」後按「執行測試」或排程「立即」；Agent 會在 5 秒內認領並自動開啟瀏覽器。
    5. 
                

                
Tip：agent 只要保持執行就會持續輪詢，不用每次重啟。要停掉 agent 按 Ctrl+C 即可。
