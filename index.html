啟動錄製器

使用方式：建立 session 後，下方已備好 Windows (PowerShell) 與 macOS / Linux (bash) 兩種一鍵指令； 依自己的作業系統複製對應那一欄，貼進終端機 Enter 即可開啟 Playwright 錄製器。 操作完成後關閉視窗，會自動產生 trace.zip 並上傳至本系統。

Windows (PowerShell) — 一鍵指令
macOS / Ubuntu / Linux (bash / zsh) — 一鍵指令

Windows：第一次若出現「執行原則」警告，在 PowerShell 中執行 Set-ExecutionPolicy -Scope CurrentUser RemoteSigned 一次即可。 macOS / Linux：需已安裝 python 與 curl（大多數系統預設內建）；若沒裝 Playwright，第一次執行會自動從 pip 安裝。 RF Browser 已內建 playwright，無需額外安裝。

上傳錄製結果

貼上 cURL 指令

用法：在 Chrome DevTools 的 Network 分頁對請求按右鍵 → Copy → Copy as cURL。可一次貼多個，以空行分隔。 解析時會自動轉換為 Http.SetHeader / Http.GET / POST / ... 步驟。

貼上 Appium Python 腳本

用法：從 Appium Inspector 錄製後匯出 Python 腳本，或直接貼上手寫的 driver.find_element(...) 操作。 解析時會自動轉換為 Mobile.Click / Input / Tap / Swipe / PressBack 等步驟。

本機 Agent 設定

            

            

                
「本機」模式會把測試交給本機的 local_agent.py 執行，會 自動開啟有頭 Chromium，你可以在本機視覺化觀察整個測試過程。

                
    
                    
  1. 點下方「下載 Agent」取得 local_agent.py。
    2. 
                    
  2. 一次性安裝依賴（在本機終端機）：
    
                        pip install playwright requests
    
                        playwright install chromium
                    
    3. 
                    
  3. 啟動 agent（保持常駐）：
                        
    
                            
                            
                        
    
                    
    4. 
                    
  4. 切換頂部「環境：本機」後按「執行測試」或排程「立即」；Agent 會在 5 秒內認領並自動開啟瀏覽器。
    5. 
                

                
Tip：agent 只要保持執行就會持續輪詢，不用每次重啟。要停掉 agent 按 Ctrl+C 即可。
